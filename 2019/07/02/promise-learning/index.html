前端异步方案-promise

背景：ajax请求如果出现callback嵌套，会造成不利于代码阅读，这就是我们常说的毁掉地狱。ECMAscript6 为开发者提供了Promise对象。

Promise的三种状态

  1. Pending 创建Promise对象实例的初始化状态
  2. Fulfilled 成功回调状态。在链式then方法执行resolve
  3. Rejected 失败回调状态。在链式catch方法中执行reject
